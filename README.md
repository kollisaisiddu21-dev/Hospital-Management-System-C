# 🏥 Hospital Management System (C)

A console-based **Hospital Management System** developed in **C** using a **Singly Linked List** for dynamic patient record management.

This project was built to strengthen my understanding of C programming, dynamic memory allocation, linked lists, and modular programming. It is an ongoing project that will continue to receive new features and improvements over time.

---

## 📌 Features

- ➕ Add a new patient
- 📋 Display all patient records
- 🔍 Search patient by ID
- 🔎 Search patient by Name
- ✏️ Update patient details
- ❌ Delete patient record
- 📊 Display hospital statistics
- ✅ Prevent duplicate Patient IDs
- 🧠 Dynamic memory allocation using Linked Lists

---

## 📝 Patient Information Stored

Each patient record contains:

- Patient ID
- Name
- Age
- Gender
- Mobile Number
- Disease
- Blood Group
